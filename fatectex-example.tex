\begin{Resumo}
Este documento exemplifica a elaboração de trabalho acadêmico (projeto de pesquisa, trabalho de conclusão de curso, dissertação e tese) a partir da classe FATEC\TeX, uma extensão da classe \abnTeX\ para a FATEC Pompéia. Além de apresentar comandos básico de \LaTeX\ para inclusão de equações, tabelas e figuras, o documento mostra como utilizar pacotes adotados pela classe FATEC\TeX\ para gerar referências bibliográficas, listas símbolos, caixas para teoremas e algoritmos, dentre outros elementos úteis ou obrigatórios para trabalhos acadêmicos. Espera-se que este documento facilite o uso da classe FATEC\TeX\ na elaboração de trabalhos de alta qualidade gráfica mesmo por usuários com pouca experiência em \LaTeX.
\end{Resumo}
% ---

% ---
% Resumo em inglês
% ---
\begin{Abstract}
This document illustrates the development of academic work (research projects, final papers, dissertations, and theses) using the FATEC\TeX class, an extension of the \abnTeX\ class for FATEC Pompéia. In addition to presenting basic \LaTeX\ commands for including equations, tables, and figures, the document demonstrates how to use packages adopted by the FATEC\TeX\ class to generate bibliographic references, symbol lists, boxes for theorems and algorithms, among other useful or mandatory elements for academic work. It is hoped that this document will facilitate the use of the FATEC\TeX\ class in the development of high-quality graphical work, even for users with little \LaTeX experience.
\end{Abstract}
